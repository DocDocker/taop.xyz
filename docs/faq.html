What is a LEFT JOIN in SQL?

SQL allows developers to manipulate relations. A JOIN is an operation that builds a new relation, dynamically, from two other relations. A LEFT JOIN builds the new relation by associating 0 or more rows from the second input relation to every row from the first relation.
